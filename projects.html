Environmental, Hackathon, and Independent Projects

These are my environmental, hacakthon, and independent projects.

Environmental Projects

Automated Water Conservation System

Using Arduino, a relay, humidity sensors, soil moisture sensors, temperature sensors, light intensity sensors, and a data logging shield, I built an automated water conservation system that watered a garden in the most environment-friendly and healthy manner.

Climate Data Logger

Using an Arduino, a data logging shield, humidity sensors, temperature sensors, and moisture sensors, I built a climate data logger. My current goal for these projects is to bring my data collection systems within the National Oceanic Atmospheric Administration’s parameters for use as official climate change data.

Hackathons

Teen Hacks 2018: Best Overall Hack- II

Built Noma, a web application that doctors and non-doctors alike can use for an early and non-invasive diagnosis of melanoma. It uses a Machine Learning algorithm along with 4 Convolutional Neural Networks and Sigmoid Activation to classify whether an image is melanoma targeted skin.

Security System

Intruder Detection System

Using Arduino, motion sensors, a speaker and other electronics, I made an intruder detection system that would ring if anyone passed by a group of motion sensors.

Face Recognition System

Using OpenCV, Open-Source C++, and Arduino, I made a face recognition system that could register and recognize faces for authentication.

I combined the two to make a DIY, rudimentary security system for my house.

Voice Command Turtle Bot in Simulation

Using ROS and C++, simulated a robot that uses voice-command to move around a simulated house. Gave it a place to go, eg. “Go to the kitchen”, and it would confirm the destination and would travel there. I plan to extend this into an assistive device for the blind or paralyzed.

Check out the code!

Voice Command Medical Personal Assistant

Using a Raspberry Pi, the Google Voice Recognition API, the Jasper Project, and Python code, I created a voice command based personal assistant that would listen to speech and would subsequently answer or execute commands based on it. Used medical API’s to create modules where the user would list out symptoms and the device would respond with a diagnosis. Also used Twilio APIs to make voice/text calls to pre-registered phone number, which includes 911, emergency contact person numbers.
